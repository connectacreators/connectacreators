**Edit A — Change nav arrows from absolute to fixed positioning.**

Find the nav arrows div at approximately line 700-716:
```tsx
<div className="absolute left-4 top-1/2 -translate-y-1/2 flex flex-col gap-2 z-10 hidden lg:flex">
```

The `absolute top-1/2 -translate-y-1/2` approach ties arrow position to the parent element's height, which changes as the reel column transforms. Replace with `fixed` positioning using viewport-relative coordinates:

```tsx
<div className="fixed left-4 top-1/2 -translate-y-1/2 flex flex-col gap-2 z-[50] hidden lg:flex">
```

Key changes:
- `absolute` → `fixed` (viewport-relative, immune to parent transforms and height changes)
- `z-10` → `z-[50]` (fixed elements need a higher z-index to sit above other stacking contexts)
