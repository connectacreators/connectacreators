Summary of changes
What gets built, in priority order.
  • 🔴
    Fix source_ref migration [1 SQL line, fixes broken comments now]
    Revision comments silently fail on multi-source videos. Adding the missing column unblocks existing functionality immediately.
  • 🟡
    Platform field per video (Reels / TikTok / Shorts / Other)
    One dropdown per row. Drives aspect ratio expectations, caption limits, and hashtag strategy downstream.
  • 🟡
    Revision round counter + Approve / Request Changes buttons
    Auto-increment round number when status is set back to revision. Replace status dropdown in review modal with two big buttons — the most impactful UX change vs. competitors.
  • 🟡
    Internal-only comments (hidden from client)
    Add visibility toggle on each comment. Internal comments show with a lock badge to team, completely hidden on public review page. Essential for agency workflows.
  • 🟢
    Version stacking (V1/V2/V3 history)
    Upload new cut as a version instead of overwriting. All versions stay accessible. Version picker in review modal. Adds version_number + parent_id columns.
  • 🟢
    Expanded status pipeline (7 stages)
    Add Internal Review, Awaiting Approval, and Approved as distinct stages between editing and publishing. Makes the handoff visible to everyone.
  • 🟢
    Review link expiry + password
    Generate time-limited signed links for client review. Optional password. Audit log of who approved and when.
